Overview

Unfurl

A global text expander for Windows. Type a short trigger word anywhere and Unfurl replaces it with your saved text instantly — in any application.

Type myemail → Unfurl deletes the trigger and pastes hello@example.com. Works in every app, instantly, with no plugins required.

Unfurl monitors your keyboard globally using a low-level hook. The moment it recognises a saved trigger word, it backspaces over it and pastes the full replacement text via the clipboard. The original clipboard contents are restored immediately after.

The trigger buffer resets on Enter, Escape, or any modifier key (Ctrl, Alt, Win, Shift), so normal keyboard shortcuts are never affected.

Getting started

Installation

Installer (recommended)

1
Download Unfurl-Setup-x.x.x.exe from the Releases page.
2
Run the installer. You can choose a custom install directory.
3
Unfurl launches automatically and appears in the system tray.

Portable

A portable .exe is also available. No installation required — run it directly. Data is stored in the same directory as the executable.

Requirements: Windows 10 or later (64-bit). Administrator privileges may be required for the portable build.
Usage

Creating & managing shortcuts

Creating a shortcut

1
Click + New in the toolbar, or press Ctrl+Alt+S to open the window.
2
Enter a trigger word. Lowercase letters and numbers only (e.g. myemail, sig1).
3
Optionally set a category to keep shortcuts organised.
4
Enter the replacement text. Supports multiple lines and dynamic variables.
5
Click Save or press Ctrl+Enter.

Editing

Click any row in the list (or the pencil icon) to open it in the editor. Saving with the same trigger word overwrites the existing entry.

Deleting

Click the trash icon on any row. A toast appears immediately with an Undo button. You can also delete all shortcuts at once from Settings → Danger Zone.

Reordering

Drag rows by the handle on the left to change their order. Order is persisted to disk.

Favourites

Click the star icon on any shortcut to add it to Favourites (maximum 10). Favourites appear in the tray icon's context menu and in the mini pill for quick one-click copy.

Searching & filtering

Use the search field in the toolbar to filter by trigger or replacement text. Use the category chips below the toolbar to filter by category.

Tip: Set a trigger prefix (e.g. ;) in Settings so you'd type ;myemail instead of myemail. This prevents accidental expansion of common words.
Usage

Dynamic variables

Include any of these placeholders in your replacement text. They are resolved at the moment of expansion.

VariableExample outputDescription
{date}27/04/2026Today's date
{time}09:30 AMCurrent time
{datetime}27/04/2026, 09:30 AMFull date and time
{day}MondayDay of the week
{month}AprilMonth name
{year}2026Current year

Variables are case-insensitive — {DATE} and {date} both work. Click a variable chip in the editor to insert it at the cursor position.

Features

Mini pill mode

A compact always-on-top floating button that sits over your other windows. Use it when you want quick access to Unfurl without keeping the full window open.

Monitoring
Paused
ActionResult
Left-click the pillReturn to the full window
Right-click the pillOpen context menu
Context menu → FavouritesCopy a favourite shortcut to clipboard
Context menu → Pause / ResumeToggle expansion on or off
Context menu → QuitExit the application

The status dot on the pill is green when expansion is active and red when paused. Toggle with Ctrl+Alt+P or the mini mode button in the title bar.

Features

App rules

Disable expansion inside specific applications. Unfurl checks the currently focused window and silently skips expansion if it is on the blocked list.

Blocking an app

1
Go to App Rules in the sidebar.
2
The left panel lists all running processes. Click any entry to block it.
3
The blocked app appears in the right panel immediately.

Unblocking an app

Find the app in the Blocked Apps list on the right and click the trash icon.

Note: Apps are identified by their executable name (e.g. chrome.exe). If an app is not in the running list, click Refresh List or use the search field to find it.
Features

Import & export

Exporting

Click Export in the toolbar. A dated .json file is downloaded containing your full shortcut library.

Importing

Click Import and select a .json file. Two formats are accepted:

Array format

[
  { "trigger": "myemail", "message": "hello@example.com", "category": "Work" },
  { "trigger": "sig",     "message": "Best regards,\nJane" }
]

Object format

{
  "myemail": "hello@example.com",
  "sig":     "Best regards,\nJane"
}

Duplicate triggers are skipped — existing shortcuts are never overwritten on import. The import summary toast shows how many were added and how many were skipped.

Shortcuts file

Shortcuts are stored in shortcuts.json in Unfurl's application data folder by default. You can change this path in Settings → Storage — for example, pointing it at a Dropbox or OneDrive folder to sync across machines.

{
  "myemail": {
    "message":    "hello@example.com",
    "usageCount": 12,
    "category":   "Work",
    "createdAt":  1714000000000
  }
}

Unfurl writes atomically via a .tmp → rename pattern and keeps a .bak backup on every write.

Auto-reload: With Auto-check JSON for changes enabled in Settings, Unfurl polls the file every 30 seconds and reloads automatically if it detects an external modification.
